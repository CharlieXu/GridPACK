38 /**
39  * This class provides a way to assemble a graph adjacency list when
40  * graph node and edge information is arbitrarily distributed. For
41  * example, one process may "own" a certain node, but the way that
42  * node is connected to others may be on another process.
43  *
44  * It just works with integral indexes. It is assumed that the global
45  * indexes start with 0, are unique, and that the largest index is N-1
46  * if there are N nodes/edges. There's probably some special term for
47  * that.
48  *
49  */
