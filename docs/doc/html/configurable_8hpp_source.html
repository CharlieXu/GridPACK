28 /**
29  * This serves as the base for any class that has run-time
30  * options. The options required by a instance of this base class are
31  * identified by a string and may have any (or one of many) types.
32  *
33  * Instances have two paths, one for default values for all instances
34  * and one for specific instances. The instance is created with the
35  * former and it cannot be changed by any public means. The latter
36  * may be created created by prepending a name or path to the default.
37  * For example, suppose a "linear solver" is a subclass, and it needs
38  * a "tolerance". The default path might be:
39  *
40  * "linear solver"/"tolerance" = 1.0e-06
41  *
42  * which would be used by all instances. Say there is "nonlinear
43  * solver" that uses a "linear solver" instance (both subclasses of
44  * Configurable), a path such as
45  *
46  * "nonlinear solver"/"linear solver"/"tolerance" = 1.0e-03
47  *
48  * would distinguish it from the default.
49  *
50  * I think this is just a wrapper on boost::property_tree.
51  *
52  */
